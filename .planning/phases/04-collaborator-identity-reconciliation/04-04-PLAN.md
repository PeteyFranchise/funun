| client → PATCH /api/collaborators/:id (archived_at) | Untrusted timestamp; the server must own the archive instant, never trust a client-supplied date |

| client → user_profiles upsert (settings) | First-time INSERT must be permitted by explicit RLS, scoped to auth.uid() = id |
