    - terms is null/undefined (project has no pre-cleared terms) → matched false with a reason naming the absent terms (D-15a routes it to admin negotiation).
    - all five dimensions satisfied → matched true, empty reasons.
    - budget_cents below terms.min_fee_cents → matched false with a fee reason.
    - a requested usage type not present in allowed_usage_types → matched false with a usage reason.
    - a requested territory not present in terms.territories → matched false with a territory reason.
    - request.exclusivity true while terms.exclusivity_allowed false → matched false with an exclusivity reason.
    - request.term_months greater than terms.max_term_months → matched false with a term reason.
    - multiple violations accumulate multiple reasons (not short-circuited).

    computeNetFee(grossCents, commissionPct):
    - exact integer cents, no floating-point drift (e.g. a 25% commission on an odd cent amount round-trips: commissionCents + artistNetCents === grossCents, always).
    - commissionPct 0 → full gross to artist, zero commission.
    - commissionPct 100 → zero to artist.
    - negative or non-finite gross or pct → throws a descriptive Error (project convention: no silent failures).
